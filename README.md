# 🏏 Cricket ODI World Cup 2023 Dashboard  
### 📊 Player Statistics & Insights 
---

## 📌 Project Overview  

This project presents an **interactive dashboard** built on the **Cricket ODI World Cup 2023 dataset**, focusing on **player performance analysis and insights**.  

The dashboard enables users to explore detailed player statistics including batting, bowling, and overall performance throughout the tournament.

---

## 🚀 Features  

- 📈 **Player Performance Analysis**  
  Analyze runs, strike rate, batting average, wickets, and economy rate  

- 🧑‍🤝‍🧑 **Player Information Section**  
  Includes player role, team, matches played, and performance summary  

- 📊 **Interactive Dashboard**  
  Dynamic charts and graphs for better visualization  

- 🏏 **Top Performers Identification**  
  - Highest run scorers  
  - Leading wicket-takers  
  - Best strike rates and economy  

- 🔍 **Comparative Analysis**  
  Compare players across different teams  

---

## 🛠️ Tech Stack  

- 📊 Streamlit  
- 🐍 Python (Pandas, NumPy,Matplotlib)   
- 📁 CSV Dataset  

---

## 📂 Dataset  

The dataset includes detailed statistics from the **ODI World Cup 2023**, such as:  
- Batting performance  
- Bowling performance  
- Match-wise data  
- Player details
